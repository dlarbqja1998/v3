# 골라바유 v3 작업 정의

## 목표

골라바유 v3는 고려대 세종 학생이 매일 확인하는 지도 기반 생활 홈이다. 첫 화면에서 식당, 학식, 셔틀 정류장, 모임 진입 정보를 바로 볼 수 있어야 한다.

## 이번 단계에서 할 일

1. `C:\v3\v3`에 새 SvelteKit 앱을 만든다.
2. 기존 v2 운영 프로젝트와 분리된 v3 코드 구조를 만든다.
3. 네이버 지도 연동 전에도 개발 가능한 지도 홈 UI를 만든다.
4. `places` 중심 도메인 타입과 seed 데이터를 만든다.
5. Cloudflare/Neon 리소스가 준비되면 붙일 수 있도록 API route 경계를 만든다.

## 이번 단계에서 하지 않는 일

1. 기존 v2 코드나 DB를 수정하지 않는다.
2. 기존 `.env` 값을 복사하지 않는다.
3. 실제 Neon `DATABASE_URL`을 문서에 남기지 않는다.
4. 네이버 지도 API를 사용자 조작마다 호출하는 구조를 만들지 않는다.

## 현재 배포 기준

```txt

```

초기 OAuth, 지도 API 허용 도메인, 분석 도구 설정은 이 URL을 기준으로 잡는다.
